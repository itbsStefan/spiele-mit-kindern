Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat. Italic To make a word or sentence italic, surround it with a * or _ symbol. Markdown Only this *word* and this _word_ should be italic. And these *two words* should be italic, as well as _these two_. Output HTML Only this word and this word should be italic. And these two words should be italic, as well as these two. Bold To make a word or sentence bold, surround it with ** or __ symbols. Markdown Only this **word** and this __word__ should be bold. And these **two words** should be bold, as well as __these two__. Output HTML Only this word and this word should be bold. And these two words should be bold, as well as these two. Link To add a link, surround the link text [ and ] symbols, followed by the link URL surrounded by ( and ) symbols. Markdown A simple [text link](http://howtomarkdown.com) mid sentence. Another link: [http://howtomarkdown.com](http://howtomarkdown.com) Output HTML A simple text link mid sentence. Another link: http://howtomarkdown.com Image To add an image, use an ! symbol followed by the image alt text wrapped in [ and ] symbols and the image URL surrounded by ( and ) symbols. Markdown ![a cat picture](http://placekitten.com/200/200) Output HTML a cat picture List A list is a block of lines, each starting with a * or incrementing number followed by a . symbol. Markdown An unordered list: * Item 1 * Item 2 An ordered list: 1. Item 1 2. Item 2 Output HTML An unordered list: Item 1 Item 2 An ordered list: Item 1 Item 2 About Markdown was created by John Gruber. For a list of all syntax options see the Markdown project page.
